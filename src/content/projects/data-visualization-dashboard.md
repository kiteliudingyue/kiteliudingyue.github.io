## Overview

This interactive dashboard provides deep insights into sales performance across different regions and product categories. Built with React and D3.js, it offers real-time data filtering and visualization capabilities.

## Key Features

- **Real-time Data Processing**: Live updates with WebSocket integration
- **Interactive Charts**: Zoomable sunburst, area charts, and heat maps
- **Advanced Filtering**: Multi-dimensional filtering by region, time, and product
- **Export Capabilities**: Download visualizations as PNG or SVG

## Technology Stack

- React 18 with TypeScript
- D3.js for custom visualizations
- Tailwind CSS for styling
- Vitest for testing

## Insights & Impact

The dashboard helped identify key sales trends leading to a 15% increase in quarterly revenue. The predictive analytics module forecasted seasonal demand patterns with 92% accuracy.
